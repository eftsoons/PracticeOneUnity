ScriptableObjects:

1. DroneConfig
   Назначение: Базовый прототип конфигурации
   Как подключить: автоматичечкий подключается
   Какие настройки можно менять: можно менять базовую структуру конфигурации
2. LanguageData
   Назначение: Базовый словарь с переводами текстов и вариантами языков
   Как подключить: автоматичечкий подключается
   Какие настройки можно менять: можно менять количество языков и сами текста переводов

Scripts:

3. AnalyticsManager
   Назначение: Сохраняет аналитику в папку /Analytics, имеет публичные методы под внесения корректировок в аналитику из игры
   Как подключить: автоматичечкий подключается
   Какие настройки можно менять: -
4. ConfigManager
   Назначение: Реализует прокладку для доступа DroneConfig из скриптов
   Как подключить: автоматичечкий подключается
   Какие настройки можно менять: подставить свой DroneConfig
5. HandleButtonClick
   Назначение: Обработчик кликов для кнопкок
   Как подключить: создать объект и навесить обработчик на него, далее прикрепить объект к кнопке и выбрать нужную функцию
   Какие настройки можно менять: -
6. HandleValueInput
   Назначение: Обработчик изменения текста для input
   Как подключить: создать объект и навесить обработчик на него, далее прикрепить объект к input и выбрать нужную функцию
   Какие настройки можно менять: -
7. LocalizationManager
   Назначение: Быть прокладкой между скриптами и LanguageData и переводить текст с помощью методов
   Как подключить: автоматичечкий подключается
   Какие настройки можно менять: -
8. LocalizationText
   Назначение: Автоматически переводить текст на котором весит данный скрипт (слово для перевода берет прямо из текста)
   Как подключить: повесить скрипт на объект текста
   Какие настройки можно менять: -
9. SaveData
   Назначение: Сохранять итоговые данные в папку /Saves
   Как подключить: автоматичечкий подключается
   Какие настройки можно менять: можно настроить версию сохранения
